3. If you see it, tap to launch!

### Option 2: Check Terminal
Look for these messages:
- ✅ `** BUILD SUCCEEDED **` - Build completed!
- ✅ `Installing...` - App installing
- ✅ `Launching...` - App launching
- ❌ `** BUILD FAILED **` - Build failed (check errors)

### Option 3: Check Metro Logs
In Metro terminal, look for:
- ✅ `Bundle loaded successfully` - App loaded
- ❌ Red error messages - Something wrong
